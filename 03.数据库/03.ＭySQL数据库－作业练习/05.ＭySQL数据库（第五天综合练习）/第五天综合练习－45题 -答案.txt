10、查询Score表中的最高分的学生学号和课程号。（子查询或者排序）

	排序：按照degree列降序排列，用limit子句取第一条记录
	SELECT sno,cno  FROM score ORDER BY degree DESC LIMIT 1;
	子查询：通过一个子查询找出score表中degree列的最大值，然后作为一个条件与degree进行比较
	SELECT sno,cno FROM score 
	WHERE degree=(SELECT MAX(degree)FROM score);
	
11、查询每门课的平均成绩。

	单表：分组+聚合函数
	SELECT cno,AVG(degree) FROM score GROUP BY cno;
	多表显示课程名称：连接+分组+聚合函数。score表与course表有共同的列cno,可通过cno作为关键子连接，然后分组+聚合函数，使用共同的列必须指定表名。
	SELECT course.cno,cname,AVG(degree) FROM course
	JOIN score ON course.cno=score.cno
	GROUP BY course.cno;
	
12、查询Score表中至少有5名学生选修的并以3开头的课程的平均分数。
	1.通过where+like去除多余的记录
	2.cno进行分组+having+聚合函数
	
	SELECT cno,AVG(degree) FROM score 
	WHERE cno LIKE "3%"
	GROUP BY cno
	HAVING COUNT(*)>5;

13、查询分数大于70，小于90的Sno列。
	
	不能用between and,因为between and是闭区间
	SELECT sno FROM score WHERE degree>70 AND degree<90;

14、查询所有学生的Sname、Cno和Degree列。

	两表连接，sname在student表，cno和degree在score表
	连接关键字：两个表都有的一个列sno
	SELECT sname,cno,degree FROM student
	JOIN score
	ON score.sno=student.sno;
